# New Scenario Checklist

Steps to add a new situational playbook entry.

## Creating the entry

- [ ] Copy `playbook/_template.html` to `playbook/[situation-slug].html`
- [ ] Fill in all four sections: Situation (1 sentence), Say This, Steps (3), Don't (2-3)
- [ ] Update the `<title>` tag and `<h1>` to `[Situation Title]`
- [ ] Keep only the relevant principle tags (1-3)

## Linking it up (manual for now)

- [ ] For each principle tagged in the entry, open its page in `principles/` and add a link in the "Playbook entries" section
- [ ] Add the entry to `playbook/index.html`
- [ ] Update the entry count in `index.html` playbook callout badge

> **Note:** There's no automatic linking between playbook entries and principle pages — each connection has to be added by hand.
